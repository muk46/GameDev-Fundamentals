# GameDev Fundamentals

게임 클라이언트 개발자를 목표로 공부한 내용을 정리하고, 직접 구현한 실습 결과물을 모아둔 레포지토리입니다.

## 구조

```
01_GameMath/          — 게임 수학 (벡터, 행렬, 쿼터니언)
02_Cpp/               — C++ 심화 (메모리, 다형성, Modern C++)
03_Graphics/          — 그래픽스/렌더링 (파이프라인, 셰이더, 라이팅)
04_DataStructures/    — 자료구조/알고리즘 (구현 및 문제풀이)
05_UnrealEngine/      — UE 심화 (프레임워크, 네트워크, 최적화)
```
