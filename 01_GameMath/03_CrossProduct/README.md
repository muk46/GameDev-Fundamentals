# 외적 (Cross Product)

## 외적이란?

두 벡터를 넣으면 **두 벡터에 수직인 새로운 벡터**가 나오는 연산.

```
내적: A · B = 숫자     (얼마나 같은 방향인지)
외적: A × B = 벡터     (수직인 방향)
```

## 2D 외적

3D에서는 벡터가 나오지만, 2D에서는 숫자 하나로 쓸 수 있다.

```
A × B = Ax * By - Ay * Bx
```

### 내적 vs 외적 공식 비교

```
내적: Ax*Bx + Ay*By   (같은 성분끼리 곱하고 더하기)
외적: Ax*By - Ay*Bx   (교차해서 빼기)
```

## 결과값의 의미

```
양수 (+)  →  B가 A의 왼쪽

    B
    ↑
    |
    *──→ A       외적 = 양수 (왼쪽!)

0         →  평행

음수 (-)  →  B가 A의 오른쪽

    *──→ A
    |
    ↓
    B             외적 = 음수 (오른쪽!)
```

## 게임에서의 활용

### 적이 왼쪽인지 오른쪽인지 판별

```
    적(왼쪽)          적(오른쪽)
      *                     *
       \                   /
        \  외적 > 0       /  외적 < 0
    플레이어 ──→     플레이어 ──→
```

### 내적 + 외적 = 4방향 판별

| 연산 | 판별 |
|------|------|
| 내적 > 0 | 앞 |
| 내적 < 0 | 뒤 |
| 외적 > 0 | 왼쪽 |
| 외적 < 0 | 오른쪽 |

조합하면: 앞-왼쪽, 앞-오른쪽, 뒤-왼쪽, 뒤-오른쪽

```
전방: (0, 1) 위쪽

        적B (-2,1)    적A (1,3)
         앞-왼쪽  \   / 앞-오른쪽
                   \ /
            (0,0) ──↑──
                   / \
         뒤-왼쪽  /   \ 뒤-오른쪽
        적D (-3,-1)   적C (1,-2)
```

| 적 | 내적(앞/뒤) | 외적(좌/우) | 결과 |
|----|------------|------------|------|
| A (1,3) | 양수 → 앞 | 음수 → 오른쪽 | 앞-오른쪽 |
| B (-2,1) | 양수 → 앞 | 양수 → 왼쪽 | 앞-왼쪽 |
| C (1,-2) | 음수 → 뒤 | 음수 → 오른쪽 | 뒤-오른쪽 |
| D (-3,-1) | 음수 → 뒤 | 양수 → 왼쪽 | 뒤-왼쪽 |

## 3D 외적과 오른손 법칙

2D에서 외적은 숫자 하나지만, 3D에서는 **두 벡터에 수직인 새로운 벡터**가 나온다.
방향은 오른손 법칙으로 결정.

```
1. 오른손 손가락을 A 방향으로 편다
2. B 방향으로 손가락을 감아 쥔다
3. 엄지가 가리키는 방향 = 외적 결과 방향

    엄지 ↑ (A × B 결과)
    |
    *──→ A
   /
  ↙ B
```

대표적 예시:
```
오른쪽(X) × 앞쪽(Y) = 위쪽(Z)
```

게임에서의 활용:
- **법선 벡터(Normal)**: 삼각형의 두 변을 외적 → 표면에 수직인 벡터 → 라이팅 계산에 필수
- **카메라 업 벡터**: Forward × Right = Up 방향

## 코드 리뷰에서 배운 점

| 실수 | 수정 |
|------|------|
| 외적에서 `+` 사용 (내적이랑 헷갈림) | `-` 가 맞음: `Ax*By - Ay*Bx` |
| 이미 구한 `dir` 변수 안 쓰고 중복 계산 | 변수 재활용하기 |

## 퀴즈 오답 노트

- 외적 부호 계산 시 **음수 곱하기** 조심: `-(-1)` = `+1`
- 내적은 `+`, 외적은 `-` 헷갈리지 말 것
