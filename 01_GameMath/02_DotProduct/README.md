# 내적 (Dot Product)

## 내적이란?

두 벡터를 넣으면 **숫자 하나(스칼라)**가 나오는 연산.

```
A · B = Ax * Bx + Ay * By
```

예) `A = (3, 4)`, `B = (2, 1)` → `3*2 + 4*1` = **10**

## 내적의 의미

```
A · B = |A| * |B| * cos(θ)     ← θ는 두 벡터 사이 각도
```

두 벡터가 **얼마나 같은 방향인지** 알 수 있다.

```
양수 (+)  →  같은 방향 (0° ~ 90°)

    A →
    B →

0         →  수직 (정확히 90°)

    A →
    B ↑

음수 (-)  →  반대 방향 (90° ~ 180°)

    A →
    B ←
```

## 정규화된 벡터의 내적 = cos(θ)

| 내적 결과 | 각도 | 의미 |
|-----------|------|------|
| 1 | 0° | 완전히 같은 방향 |
| 0.866 | 30° | 거의 같은 방향 |
| 0.5 | 60° | 비스듬히 같은 방향 |
| 0 | 90° | 수직 |
| -0.5 | 120° | 비스듬히 반대 |
| -0.866 | 150° | 거의 반대 |
| -1 | 180° | 완전히 반대 방향 |

## 게임에서의 활용

### 1. 적이 앞에 있는지 뒤에 있는지

```
        적
        *
       /
      /  내적 > 0 → 앞에 있음!
    플레이어 ──→ 전방(Forward)

    전방 벡터 · (적 방향) > 0 → 앞
    전방 벡터 · (적 방향) < 0 → 뒤
    전방 벡터 · (적 방향) = 0 → 옆
```

### 2. 시야각 (FOV) 체크

```
          시야각 60°
         /       \
        /    *적   \
       / (시야 안!) \
      /             \
    플레이어 ──→ 전방

    시야각 60° → 반각 30° → cos(30°) ≈ 0.866
    내적 결과 > 0.866 이면 시야 안에 있음!
```

## 퀴즈 오답 노트

- 내적 값과 각도의 관계: cos(θ)를 외울 필요 없이, **양수=같은방향, 0=수직, 음수=반대** 감만 잡으면 된다
