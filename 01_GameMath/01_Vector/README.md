# 벡터 기초

## 스칼라 vs 벡터

- **스칼라**: 크기만 있는 값 (체력 100, 속도 5)
- **벡터**: 크기 + 방향 (오른쪽으로 5의 속도)

```
스칼라:  ───── 5

벡터:   ────→ (3, 4)  방향 + 크기
```

## 벡터 표기

```
2D: V = (x, y)
3D: V = (x, y, z)

UE:    FVector(X, Y, Z)
Unity: Vector3(x, y, z)
```

## 벡터의 크기 (Magnitude)

```
|V| = sqrt(x² + y²)        // 2D
|V| = sqrt(x² + y² + z²)   // 3D
```

예) `V = (3, 4)` → `sqrt(9 + 16)` = `sqrt(25)` = **5**

```
    4 |       * (3, 4)
      |      /|
      |     / |
      |  5 /  | 4        ← 피타고라스 정리
      |   /   |
      |  /    |
      | /     |
    0 *───────*
      0   3
```

## 정규화 (Normalize)

방향은 유지하고 크기를 1로 만드는 것.

```
normalize(V) = V / |V|
```

예) `V = (3, 4)` → `(3/5, 4/5)` = `(0.6, 0.8)`

```
원본 벡터:     ──────────→ (3, 4)     크기 = 5
정규화 벡터:   ──→ (0.6, 0.8)         크기 = 1
                ↑ 방향은 같고 크기만 1
```

**용도**: 방향만 필요할 때. 방향과 속도를 분리할 수 있다.

```
방향 = normalize(적 위치 - 내 위치)
이동 = 방향 * 속도
```

## 벡터 덧셈, 뺄셈

```
A + B = (Ax + Bx, Ay + By)
A - B = (Ax - Bx, Ay - By)
```

**핵심: `목표 - 출발` = 출발에서 목표를 향하는 벡터**

```
적 위치 - 내 위치 = 나에서 적 방향
내 위치 - 적 위치 = 적에서 나 방향 (반대!)
```

```
    5 |          적 (5, 5)
      |        ↗
      |      /
      |    /  방향벡터 (3, 4)
      |  /
    1 | 나 (2, 1)
      |
    0 *──────────
      0  1  2  3  4  5
```

## 스칼라 곱

```
V * s = (x * s, y * s)
```

방향은 같고 크기만 바뀜.

```
V * 1:  ────→
V * 2:  ────────→       (같은 방향, 2배 길이)
V * -1: ←────           (반대 방향)
```

```
이동량 = 방향벡터 * 이동속도 * DeltaTime
```

## 이동 공식 (중요!)

```
1. 방향 벡터 = 목표 위치 - 현재 위치
2. 정규화 = 방향 벡터 / |방향 벡터|
3. 이동량 = 정규화 * 속도
4. 새 위치 = 현재 위치 + 이동량
```

```
몬스터 (0,0)                    플레이어 (6,8)
    *─────────────────────────→ *

    1) 방향 = (6,8) - (0,0) = (6,8)
    2) 크기 = sqrt(36+64) = 10
    3) 정규화 = (0.6, 0.8)
    4) 이동량 = (0.6, 0.8) * 속도10 = (6, 8)
    5) 새 위치 = (0,0) + (6,8) = (6, 8)
```

**주의: 정규화를 빼먹으면 거리에 따라 속도가 달라진다!**

## 실습: 몬스터 이동 시뮬레이션

→ [vector_move.cpp](vector_move.cpp)

```
몬스터 (3,1)                    플레이어 (7,4)
    *───────────────────────→ *

    1) 방향 벡터 = (7,4) - (3,1) = (4, 3)
    2) 크기 = sqrt(16+9) = 5
    3) 정규화 = (0.8, 0.6)
    4) 이동량 = (0.8, 0.6) * 속도5 = (4, 3)
    5) 새 위치 = (3,1) + (4,3) = (7, 4)  ← 1초만에 도착!
```

정규화(Normalize), 스칼라 곱(Scale), 벡터 덧셈(Add) 직접 구현.
이동 공식 4단계를 코드로 완성.

### 정규화 구현 시 주의점

```cpp
// 크기가 0인 벡터는 나눌 수 없음 → 예외 처리 필요
if (mag == 0) return {0, 0};

// 실무에서는 float 비교 시 엡실론 사용이 더 안전
if (mag < 0.0001f) return {0, 0};
```

float은 부동소수점이라 정확히 0이 아닐 수 있기 때문.

## 코드 리뷰에서 배운 점

| 실수 | 수정 |
|------|------|
| `From - To` (뺄셈 순서 반대) | `To - From` (목표 - 출발) |
| `int distance` (소수점 잘림) | `float distance` |
| 삼항 연산자로 긴 출력문 | `if/else`로 가독성 개선 |

## 복습 퀴즈 오답 노트

- 뺄셈 순서 주의: "A에서 B를 향한" = `B - A` (목표 - 출발)
- 이동 시 반드시 정규화 먼저, 그 다음 속도 곱하기
- 거리/크기 계산할 때 `int` 쓰지 말고 `float` 쓰기
- 새 위치 = 현재 위치 + 이동량 (이동량만 구하고 끝내지 말 것)
