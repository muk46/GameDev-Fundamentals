# 벡터 기초

## 스칼라 vs 벡터

- **스칼라**: 크기만 있는 값 (체력 100, 속도 5)
- **벡터**: 크기 + 방향 (오른쪽으로 5의 속도)

## 벡터 표기

```
2D: V = (x, y)
3D: V = (x, y, z)

UE:    FVector(X, Y, Z)
Unity: Vector3(x, y, z)
```

## 벡터의 크기 (Magnitude)

```
|V| = sqrt(x² + y²)        // 2D
|V| = sqrt(x² + y² + z²)   // 3D
```

예) `V = (3, 4)` → `sqrt(9 + 16)` = `sqrt(25)` = **5**

## 정규화 (Normalize)

방향은 유지하고 크기를 1로 만드는 것.

```
normalize(V) = V / |V|
```

예) `V = (3, 4)` → `(3/5, 4/5)` = `(0.6, 0.8)`

**용도**: 방향만 필요할 때. 방향과 속도를 분리할 수 있다.

```
방향 = normalize(적 위치 - 내 위치)
이동 = 방향 * 속도
```

## 벡터 덧셈, 뺄셈

```
A + B = (Ax + Bx, Ay + By)
A - B = (Ax - Bx, Ay - By)
```

**핵심: `목표 - 출발` = 출발에서 목표를 향하는 벡터**

```
적 위치 - 내 위치 = 나에서 적 방향
내 위치 - 적 위치 = 적에서 나 방향 (반대!)
```

예) 내 위치 `(2, 1)`, 적 위치 `(5, 5)` → `(5-2, 5-1)` = `(3, 4)`

## 스칼라 곱

```
V * s = (x * s, y * s)
```

방향은 같고 크기만 바뀜.

```
이동량 = 방향벡터 * 이동속도 * DeltaTime
```

## 이동 공식 (중요!)

```
1. 방향 벡터 = 목표 위치 - 현재 위치
2. 정규화 = 방향 벡터 / |방향 벡터|
3. 이동량 = 정규화 * 속도
4. 새 위치 = 현재 위치 + 이동량
```

**주의: 정규화를 빼먹으면 거리에 따라 속도가 달라진다!**
